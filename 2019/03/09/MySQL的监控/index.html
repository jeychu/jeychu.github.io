2019-03-09 02:05  

MySQL

监控哪些内容？

  • 数据库服务的可用性
  • 数据库性能
  • 主从同步的情况
  • 服务器的资源

怎么监控？

调优

先找出瓶颈所在，然后针对瓶颈做出优化

如何找瓶颈？

利用top ps vmstat iostat netstat /proc/

然后用perf strace ltrace systemtap 追踪进程 定位函数

zabbix添加自定义监控项

写脚本 -> 在agentd.conf中添加UserParameter -> 在server中用zabbix-get测试 ->在web添加item和graph
